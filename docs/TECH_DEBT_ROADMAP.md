## Consciously skipped (decided, not forgotten)

Re-open only if the trigger occurs:

- **PR preview deploys** ‚Äî solo GitHub-Pages portfolio; dev server + visual pipeline cover review needs. Trigger: a second regular contributor.
- **License compliance checks** ‚Äî no distribution/compliance exposure. Trigger: commercial use.
- **Commit lint / conventional commits** ‚Äî squash-merge PR titles carry the history. Trigger: automated changelog need.
- **jscpd duplication gate** ‚Äî Knip + guardrail tests catch the harmful subset; the nightly advisory clone-count ratchet (`npm run check:jscpd-ratchet`, baseline in [`jscpd-baseline.json`](jscpd-baseline.json)) surfaces growth without blocking. Trigger for a blocking gate: duplication-caused bug class appearing twice.
- **knip-ignore shrink** ‚Äî low value until the ignore list blocks a real detection.
- **Automated LLM guidance-eval harness** (Cursor SDK) ‚Äî manual protocol in [`GUIDANCE_EVALS.md`](GUIDANCE_EVALS.md) is sufficient at current cadence. Trigger: quarterly manual runs get skipped twice, or guidance regressions recur.
- **Full-suite blocking visual on PRs** ‚Äî scoped visual is blocking; full stays advisory. Flip trigger: ~2 weeks of clean nightly visual runs ([`VISUAL_REGRESSION_AGENT.md`](VISUAL_REGRESSION_AGENT.md)).
