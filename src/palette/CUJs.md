# Palette Generator — critical user journeys

## CUJ-001 — App boot

Load `/palette/`, see full-bleed random palette with thumbnail strip.

## CUJ-002 — Image sampling

Upload image(s), browse thumbnail proposals, copy hex from stripes.

## CUJ-003 — Seed harmonies

Pick seed color, generate harmony suggestions, switch thumbnails.

## CUJ-004 — Random + profiles

Shuffle random palettes; adjust mood preset and lightness/chroma bounds.

## CUJ-005 — Share palette link

Copy share link from the toolbar; open it in a new tab and see the same colors (including image-sourced palettes).
