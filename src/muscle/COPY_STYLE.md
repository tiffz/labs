# Muscle Memory — copy style

Extends [`docs/USER_COPY_STYLE.md`](../../docs/USER_COPY_STYLE.md).

## Voice

Encouraging coach, not clinician. Frame study as **reps**, **warmup**, and **deck mastery**.

## Preferred terms

| Use            | Avoid                        |
| -------------- | ---------------------------- |
| Explore        | Warmup, browse mode          |
| Active Reps    | Test mode, exam              |
| Reps remaining | Cards left, questions        |
| Clean rep      | Correct answer, pass         |
| Structure      | Anatomical structure (in UI) |

## Gatekeeper copy

One sentence, front-loaded: what is locked and the next unlock action.

Example: `Master 3 reps on core bones and joints in Fundamentals first.`

## Quiz feedback

- Correct: short (`Clean rep. Keep the form.`)
- Incorrect: point to review card, not blame (`Review the shape and try the next rep.`)
