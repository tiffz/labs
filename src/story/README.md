# Save the Cat! Story Generator

A creative writing tool that generates random story plots based on the beat structure from "Save the Cat! Writes a Novel" by Jessica Brody. Perfect for brainstorming story ideas and learning the Save the Cat! methodology.

## Features

- **10 Story Genres**: Choose from Blake Snyder's 10 story types (Whydunit, Rites of Passage, Institutionalized, etc.)
- **10 Universal Themes**: Select themes like Forgiveness, Love, Acceptance, Faith, etc.
- **Dynamic Story DNA**: Generates hero, flaw, nemesis, and settings tailored to your selections
- **15-Beat Structure**: Complete beat-by-beat story outline following the Save the Cat! methodology
- **Genre-Specific Elements**: Each genre includes unique structural elements (e.g., "The Detective" for Whydunit)
- **Reroll System**: Click the dice icon on any element to generate a new random suggestion
- **Tooltips**: Hover over help icons (?) to learn about each story element

## How to Use

1. **Select Genre**: Choose a story genre from the options
2. **Select Theme**: Choose a universal theme
3. **Generate Story**: The app automatically generates a complete story outline
4. **Reroll Elements**: Click the dice icon on any element to generate a new suggestion
5. **Explore Beats**: Review the 15-beat structure to understand story progression

## Story Structure

The generator creates stories following the Save the Cat! 15-beat structure:

- **Act 1**: Opening Image, Theme Stated, Setup, Catalyst, Debate
- **Act 2**: Break into Two, B Story, Fun and Games, Midpoint, Bad Guys Close In, All Is Lost, Dark Night of the Soul
- **Act 3**: Break into Three, Finale, Final Image

Each beat includes specific story elements tailored to your selected genre and theme.
