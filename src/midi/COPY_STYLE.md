# Midi Scratchpad — copy style

Follow [`docs/USER_COPY_STYLE.md`](../../docs/USER_COPY_STYLE.md).

## Voice

- Notebook, not DAW manual.
- Short sentences; front-load the action.

## Primary verbs

| Surface | Label                                  |
| ------- | -------------------------------------- |
| Capture | Capture last N bars                    |
| Loop    | Play loop / Stop loop                  |
| Guide   | Start guide / Stop guide               |
| Export  | Export (popover); Drag to Logic (chip) |

## Empty states

One line + what to do next. Example: "Play along with the metronome, then capture the last few bars."
