# Scrapboard — critical user journeys

## CUJ-001 — Generate layout gallery from panel count

Load `/scrapboard/`, set panel count (stepper in header), browse layout thumbnails ranked conventional → experimental, pick a layout. The full page mockup stays visible in the stage. Layouts respect trim quiet zone; full-bleed variants appear only when the generator opts in (e.g. Randomize all).

## CUJ-002 — Add panel copy (dialogue + captions)

Select a panel (click on the page or P1…Pn tab). Add dialogue blocks per character (A/B/C), captions before or after dialogue, and optional SFX. Speech bubbles auto-arrange by character.

## CUJ-003 — Print-aware export

Expand **Page & export settings** for trim, bleed, DPI, and trim guides; export PNG at configured resolution from the header. Panel gutters regenerate when print spec changes.

## CUJ-004 — Randomize

**Randomize copy** shuffles dialogue/captions only. **Randomize all** also picks panel count, layout (including angular / slant / diagonal shapes), and trim preset — occasionally a full-bleed hero layout.

## CUJ-005 — Palette tint

Paste a hex row, Coolors URL, or Labs Palette Generator share link; apply tints to the mockup.
